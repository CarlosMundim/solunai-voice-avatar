## Edge TTS - START HERE

```python
import edge_tts
import asyncio

async def speak_japanese():
    text = "こんにちは、ソルナイへようこそ"
    voice = "ja-JP-NanamiNeural"
    communicate = edge_tts.Communicate(text, voice)
    await communicate.save("output.mp3")

asyncio.run(speak_japanese())
```

Install: `pip install edge-tts`
